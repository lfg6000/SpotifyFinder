Playlist Tab

Overview

  • the playlists selected on this tab determines what playlists are examined by the 'Duplicates Tab', 'Search Tab', 'Tracks Tab', 'Artists Tab'.
  • displays all the playlists you have added to your Spotify account
  • by default all the playlists that have your User Id are at the top of the table
  • by default the first 4 playlists that have your User Id are automatically selected
  • use the Save button to save your playlist selections for your next visit

Select Playlist by User Id (drop down combo box)

  • selects playlists based on User Id
  • playlists you have created will have your User Id
  • playlists that you "Saved to Your Library" will have the User Id of user that created the playlist
  • playlists that you "Saved to Your Library" created by Spotify will have a User Id = Spotify

Delete (button)

  • delete a playlist from your spotify library.
  • only one playlist can be selected when pressing delete.

Select All (button)

  • every playlist will be selected regardless of the user id

Clear All (button)

  • every playlist will be deselected regardless of the user id

Finding a Playlist

  • click in the search box at the bottom of PlaylistName column and then type the first few letters of the playlist name

Sorting (column headers)

  • you can click on any column header to sort the table
  • to return to the original sort order press the restore button

Restore (button)

  • will return the table to the original sort order
  • if you changed a playlist in the Spotify App: press 'Reload from Spotify' button to pull in the changes

Reload from Spotify (button)

  • reloads all of your playlists from Spotify
  • reloading your playlists can be useful if you change any of your playlists in the Spotify App while using this website

Save (button)

  • Remember the playlists you have selected for your next visit
  • Each time you visit spotifyFinder.com the saved playlists will be selected by default
  • If you clear your browser cookies you will have to re-save your 'favorite' playlists
