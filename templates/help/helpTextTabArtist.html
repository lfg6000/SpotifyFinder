Artist Tab

Overview

  • displays a list of all the artists found in your selected playlists
    • add/remove selected playlists on the Playlist Tab
  • displays all the tracks for the selected artist found in your selected playlists
  • remove a track from specific playlist
  • move a track to a specific playlist
  • copy a track to a specific playlist

Finding an Artist

  • click in the search box at the bottom of ArtistName column and then type the first few letters of the artist's name
  • you can also use the Search Tab to search for an artist or track or ablum in your playlists

Remove Selected Tracks (button)

  • tracks with a checkmark present will be removed from the associated playlist
  • you can not remove tracks from a playlist you do not own

Move Selected Tracks (button)

  • tracks with a checkmark present will be moved to the destination playlist
  • you can not move tracks from a playlist you do not own
  • the destination playlist drop down box shows playlists selected on the Playlist Tab that you own
  • you can only move tracks to playlists you own

Copy Selected Tracks (button)

  • tracks with a checkmark present will be copied to the destination playlist
  • you can copy tracks from a playlist you do not own
  • the destination playlist drop down box shows playlists selected on the Playlist Tab that you own
  • you can only copy tracks to playlists you own

Sorting (column headers)

  • you can click on any column header to sort the table
  • to return to the original sort order press refresh button at the bottom of the page

Clear (button)

  • clears all selections and returns the table to the original sort order
  • refreshing your track table can be useful if you changed the column sort and want to revert to the original sorting
  • if you changed a playlist in the Spotify App: press 'Reload from Spotify' button to pull in the changes

Null Artist Id's

  • artist's with null artists ids are not displayed on the artist tab (they will be shown in the tracks tab)
  • null artist ids can occur if spotify lost the rights to an artist's work

Reload from Spotify (button)

  • reloads all of your playlists from Spotify
  • reloading your playlists can be useful if you change any of your playlists in the Spotify App while using this website
